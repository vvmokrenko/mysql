Задача:

Реализация БД для сайта booking.com


Описание файлов:

1_description.txt - общее текстовое описание БД и потенциально решаемых ею задач

2_tables.sql - скрипты для создания таблиц

3_fk.sql - скрипты для создания внешних ключей

4_fill.sql - скрипты наполнения БД данными

5_erd.png - ERDiagram для БД

6_ind.sql - скрипты для создания индексов
Примечания:
 1) Таблицы regions, addr_types, properties, booking_types, booking_properties, target_types заполняем самостоятельно.
 1) Содержимое остальных таблиц заполняем, используя сервис http://filldb.info
 2) Перед запуском скрипта 6_fill.sql, сгенерированного filldb, установим для простоты отмену проверки FK через SET FOREIGN_KEY_CHECKS = 0.
    Это даст возможность пересоздавать таблицы без предварительного удаления внешних ключей.
    На практике в случае необходимости проверки входных данных на ссылочную целостность, лучше так не делать.

7_obj.sql - скрипты функций, используемых в представлениях, представлений.

8_proc.sql - скрипты для хранимых процедур и триггеров

9_queries.sql - скрипты характерных выборок (включающие группировки, JOIN'ы)
