Booking.com

Адрес: www.booking.com

Описание:
Cистема интернет-бронирования отелей, которая предлагает самый большой выбор вариантов проживания: 
от апартаментов, домов для отпуска и маленьких семейных отелей до роскошных 5-звездочных курортов. 
При этом система включает в себя не только поиск и предложение непосредственно жилья, но и услуги по
бронированию авиабилетов, аренде машин, предложению различных вариантов досуга (например, подбор экскурсий), 
трансферов от/до аэропорта.

Перечень бизнес-процессов и свойств системы. 
    1. Арендодатель размещает свое предложение на сайте.
    2. Арендодатель может предлагать дополнительные услуги
    3. Арендатор ищет подходящие ему объекты и условия по ним. Осуществляет бронирование(покупку аренды).
    4. Существует несколько типов основных предложений(объектов бронирования) - жилье, аренда машин, авиабилеты, варианты досуга, трансферы.
    5. Каждое предложение(объект бронирования) имеет свой основной адрес.
    6. Каждый тип предложения характеризуется своим доступным набором свойств (опций, услуг, характеристик).
    7. Одинаковые свойства могут быть доступны для разных типов преложений.
    8. Свойства могут быть обязательными для конкретного предложения и опциональными. Например, кол-во комнат для жилья является обязательным, для хостела - необязательным.
    9. Арендатор может заказывать при бронировании предложения только необходимый ему набор необязательных опций(свойств, услуг).
    10. После выбора набора опций в основной форме заказа сразу отображается сумма всех выбранных опций.
    11. Арендатор может оставить свой отзыв о конкретном адресе бронирования, об услуге, о конкретном предложении по бронированию.
    12. Арендодатель может оставить отзыв об арендаторе.
    13. Для объекта важным атрибуом является удаленность от центра, расположение рядом с метро.
    14. Обязательный атрибут любого фильтра - цена.


Перечень бизнес-артефактов, которые должны или могут быть определены в системе
    1. Пользователь
    2. Арендатор
    3. Арендодатель
    4. Объект аренды(дом, квартра, комната, гостиница, хостел)
    5. Адрес объекта
    6. Рейтинг объекта
    9. Дата начала аренды
    10. Срок окончания аренды.
    11. Услуги.
    12. Удобства.
